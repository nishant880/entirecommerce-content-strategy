# Business brief: {Your brand}

> This file is the business brief Claude Code loads automatically on every session start. It grounds every playbook you run from this folder in your brand's specifics.
>
> **Two ways to populate it:**
>
> **Option A (fastest, recommended).** Leave this file as-is (or delete it entirely). Run the Content Strategy master prompt and Claude will interview you conversationally, ask every question below, and fill the file for you as you go. Takes about 10 minutes. You answer in plain language; Claude writes the structured brief.

> **Option B (DIY).** Fill in every `{placeholder}` yourself before running the playbook. Keep it tight: one or two sentences per field. Founder voice beats marketing copy.
